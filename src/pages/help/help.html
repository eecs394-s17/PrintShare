What is PrintShare?

PrintShare is a mobile app that allows you to quickly print documents on the go. Need a document printed on your way to a class or presentation? Use PrintShare to request the file be printed and ready for pickup enroute to your destination.

How do I use PrintShare?

On the main app page, click the file icon to select the file that you need printed. Then, fill out an address that you would like the document delivered to, and the time and date for delivery. Hit 'Continue' to confirm the estimated cost for the print and delivery and viola you're ready!
